left카메라는 110cm에서 아래를 기준으로 40도
+ 차량 바라보는 거에서 23도 더 틀어져 있음
따라서 차량 후면을 바라보는 영역은 157도임(차량 영역은 보지 못함)

rear카메라는 105cm에서 아래를 기준으로 50도로(카메라는 170도 범위까지 볼 수 있음)

카메라의 위치는 left는 (200,270), rear는 (300,540)이고
주차할 때 위치해야 할 곳은 (300,600)에서 rear를 바라보고 있는 상태
출차할 때 위치해야 할 곳은 (150,360)에서 left를 바라보고 있는 상태

left카메라와 rear카메라는 휠체어의 감지가 동기화 되지 않음.
따라서 주차를 할 때는 보통 left에서 출발하고 rear가 감지되면 바로 rear에게 모든 권한을 할당
출차를 할 때는 보통 rear에서 출발하고 left가 감지되면 바로 left에게 모든 권한을 할당
후진은 최소화(앞에 초음파 센서를 장착했기 때문)
휠체어의 정면 혹은 후면의 각도랑 휠체어의 최단거리랑 선을 그을 때 각도가 일정 각도 이상이 되면(약 70도) 카메라에 마커가 안 보일 수 있어서, 이동할 때 지양을 하나
어쩔 수 없이 360도 돌아서 id 0번이 보이는 것을 1번이 보여야 한다면 보일때까지 회전을 시켜야 함

우리가 보낼 신호는 F(전진), B(후진), R(오른쪽회전), L(왼쪽회전)
