## 휠체어 주차 시스템 카메라 파라미터 시트

### 1. 공통 광학 데이터 (Intrinsic Data)

두 카메라 모두 동일한 렌즈 모델을 사용하므로, 기초 캘리브레이션 값은 공유합니다.

| 항목 | 변수명 | 값 (Value / Matrix) | 비고 |
| --- | --- | --- | --- |
| **Camera Matrix** | `K` | `[[601.719, 0, 630.477], [0, 601.345, 367.212], [0, 0, 1]]` | 렌즈 초점거리 및 중심점 |
| **Distortion** | `D` | `[-0.18496, 0.02541, -0.01068, 0.00322]` | Fisheye 왜곡 계수 |
| **Marker Size** | `size` | **0.25** (25cm) | ArUco 마커 실제 크기 |
| **RMS Error** | `rms` | **0.5898** | 시스템 신뢰도 (양호) |
